# FoodAnalyzer

---

FoodAnalyzer is a Java application for local food analysis using the USDA FoodData Central API.
## Features

- **Barcode scanning** from image files
- **Food search** by barcode or product name
- **Nutrition report extraction** - nutrients, ingredients, calories
- **Local file-based caching**
- **HTTP client for FoodData Central** with DTO models
- **Logging to files** of errors and exceptions
- **Unit tests** for commands, storage, HTTP client, and parsing

## Technologies

- Java 17+
- Maven
- ZXing
- JSON processing
- JUnit 5
